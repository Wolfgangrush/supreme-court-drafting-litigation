# Privacy Policy

The Wolfgang Rush plugin family — including this plugin — operates under a single, canonical privacy policy maintained at the publisher's marketplace repository.

**Canonical Privacy Policy:**
https://github.com/Wolfgangrush/wolfgang-rush-marketplace/blob/main/PRIVACY.md

Please refer to that document for the authoritative statement on data collection (none), data flow (local execution + Anthropic Claude API only), source-code transparency (MIT licensed, fully open), and professional-confidentiality considerations for advocates and law-firm users.

---

**Plain summary** (non-binding — the canonical document linked above governs):

1. This plugin does not collect, store, or transmit any user data to the Publisher.
2. The plugin runs locally inside the user's Claude Code installation.
3. User case-folder contents flow only between the user and Anthropic's Claude API in the normal course of Claude Code operation, per Anthropic's Privacy Policy: https://www.anthropic.com/legal/privacy
4. The Publisher operates no server, telemetry, analytics service, or data-collection infrastructure of any kind in connection with this plugin.
5. Users handling privileged client material should make supply-of-content decisions consistent with Section 132 of the Bharatiya Sakshya Adhiniyam 2023 / Section 126 of the Indian Evidence Act 1872 and applicable bar-council confidentiality rules.
